# Face Recognition using CNN

## Project Overview
This project implements a Face Recognition system using a Convolutional Neural Network (CNN).

## Features
- Face image preprocessing
- CNN model training
- Face prediction
- Model evaluation

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Google Colab

## Accuracy
69.87%

## How to Run
1. Open the notebook in Google Colab.
2. Install required libraries.
3. Run all cells.
4. Train the model.
5. Test predictions.
